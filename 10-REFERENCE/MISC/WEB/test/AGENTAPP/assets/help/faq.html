FAQ

Forgot PIN?

If you forgot your PIN, please call Mobile Money Careline number at <%PI_CUST_SERV_PH_NBR%> to request a new PIN. A temporary PIN will be generated and sent to you via SMS. You can then use the temporary PIN to create a new PIN.


What if I change my mobile number?

If you wish to change your mobile number, please call Mobile Money Careline number at <%PI_CUST_SERV_PH_NBR%> for more details.


How do I change my PIN?

Visit 'My Account' in the main menu of the application and select the 'Change PIN' option. Enter the current PIN that you are using and set a new PIN.


Error in transaction processing?

Visit 'My Account' in the main menu of the application and select the 'History' option. You can view the history of your past transactions along with the transaction details and status.Alternatively, you can call Mobile Money Careline number at <%PI_CUST_SERV_PH_NBR%> to know the details of your transaction.
